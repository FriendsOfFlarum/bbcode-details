# Flarum Details BBCode

Details BBCode for Flarum.

![pages_front]

## Installation

Run following command via SSH while you're in Flarum root folder:

```bash
composer require sijad/flarum-ext-details
```

## Usage

```text
[details=Summary]Hidden Content[/details]
```
